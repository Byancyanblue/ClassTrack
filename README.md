# ClassTrack: Solusi Terpusat Pengelolaan Jadwal Perkuliahan Program Studi Informatika

## ğŸš€ Deskripsi Proyek
**ClassTrack** adalah aplikasi Proyek Akhir yang bertujuan untuk menyederhanakan pengelolaan jadwal perkuliahan di lingkungan akademik. Aplikasi ini menyediakan sistem terpusat yang efektif untuk mengatasi masalah umum seperti bentrok waktu dan kebingungan informasi jadwal di antara admin, dosen, dan mahasiswa.

ClassTrack hadir untuk menjamin transparansi dan efisiensi dalam penyebaran informasi akademik, memungkinkan semua pihak untuk mengelola dan melihat jadwal secara *real-time*.

## ğŸ¯ Tujuan Aplikasi
Aplikasi ClassTrack dirancang untuk:
1.  Menyediakan sistem terpusat untuk pengelolaan jadwal kuliah Program Studi Informatika.
2.  Mempermudah mahasiswa untuk melihat jadwal perkuliahan harian beserta informasi ruang, waktu, dan dosen pengampu.
3.  Memudahkan dosen dalam memperbarui informasi kelas (seperti kelas kosong, penggantian jadwal, atau perkuliahan daring).
4.  Memfasilitasi admin dalam melakukan CRUD (Create, Read, Update, Delete) data jadwal, mata kuliah, dan dosen.
5.  Menjamin transparansi dan efisiensi dalam penyebaran informasi akademik.

## âœ¨ Fitur Utama (Key Features)
*   **Pengelolaan Data Admin:** Mempermudah admin dalam mengelola data jadwal, mata kuliah, dan dosen.
*   **Akses Dosen:** Memberikan kemudahan bagi dosen untuk memperbarui atau melihat jadwal perkuliahan mereka.
*   **Akses Cepat Mahasiswa:** Menyediakan akses cepat bagi mahasiswa untuk melihat jadwal kuliah sesuai semester dan kelas.
*   **Pencegahan Konflik:** Mengurangi potensi kesalahan dan bentrokan jadwal antar kelas.
*   **Peningkatan Efisiensi:** Meningkatkan efisiensi dan transparansi sistem akademik kampus.
